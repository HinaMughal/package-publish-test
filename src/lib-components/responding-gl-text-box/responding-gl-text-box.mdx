# `Responding gl text box`

Responding Gl is text box field which identifies GL for responding branch for maintaining financial data.

## Validation 
- Field is Mandatory
- Field is Disabled
- Maximum Data length is 18
- Field is alphaNumericSpecial
- Field is Left Aligned

## Events Description

### RespondingGlTextBox-onChange

### RespondingGlTextBox-onKeyUp

The onkeypress event occurs when the user presses a up-arrow key (on the keyboard).
