# `Contra account number text box`
 
Contra Account Number text box allows user to enter contra account number corresponding to a `GL Account`.

## Validation 
- Field is not mandatory
- Field should be masked as (####-######-##-#)
- Max length should be 16 (including dashes)

## Events Description

Note: `None of events will occur when component is disabled`

### ContraAccountNumberTextBox-onChange

The onChange event occurs when the user change in text box

### ContraAccountNumberTextBox-onBlur

### ContraAccountNumberTextBox-onKeyPress

The onkeypress event occurs when the user presses a key (on the keyboard).

### ContraAccountNumberTextBox-onKeyUp

The onkeypress event occurs when the user presses a up-arrow key (on the keyboard).

The onfocus event occurs when an element gets focus.The onfocus event is most often used with input and select.
