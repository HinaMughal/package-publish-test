# `Narration2 text box`
 
Narration2 is a text box component that defines detailed descriptions for financial transactions to be used for reporting purpose.

## Validation 
- Field is mandatory
- Field is Editable based on screen behaviour
- Color is changed based on screen behaviour
- Text input is left Aligned
- Maximum Data length is 15

## Events Description

### Narration2TextBox-onChange

The onChange event occurs when the user change in text box

### Narration2TextBox-onBlur

The onBlur event occurs before focus lost.

### Narration2TextBox-onKeyPress

The onkeypress event occurs when the user presses a key (on the keyboard).

### Narration2TextBox-onKeyUp

The onkeypress event occurs when the user presses a up-arrow key (on the keyboard).

### Narration2TextBox-onFocus

The onfocus event occurs when an element gets focus.The onfocus event is most often used with input and select.
