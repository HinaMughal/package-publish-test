# `Card number text box`

Card number text box allows user to enter his/her `credit` card number without spaces and dashes.

## Validation 
- This field is mandatory
- Max length should be 16 (without spaces)

## Events Description

### CardNumberTextBox-onChange

The onChange event occurs when the user change in text box

### CardNumberTextBox-onBlur

This event are not being used at the moment.

### CardNumberTextBox-onKeyPress

The onkeypress event occurs when the user presses a key (on the keyboard).

### CardNumberTextBox-onKeyUp

The onkeypress event occurs when the user presses a up-arrow key (on the keyboard).

### CardNumberTextBox-onFocus

The onfocus event occurs when an element gets focus.The onfocus event is most often used with input and select.
