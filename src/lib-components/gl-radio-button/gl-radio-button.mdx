## Events Description

### GlRadioButton-onChange

The onChange event occurs when the user select any option

### GlRadioButton-onClick

This event is used to perform some action when the button is clicked.

### GlRadioButton-onBlur

The onBlur event occurs before focus lost.

### GlRadioButton-onFocus

The onfocus event occurs when an element gets focus.The onfocus event is most often used with input and select.
