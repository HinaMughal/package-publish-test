# `Transaction code text box`

 
Transaction Codes is Text Box component which defines the set up level code proposed for each financial transaction entry

## Validation 
- Max length is 5
- Field Should be numeric

## Events Description

### TransactionCodeTextBox-onChange

The onChange event occurs when the user change in text box

### TransactionCodeTextBox-onBlur

The onBlur event occurs before focus lost.

### TransactionCodeTextBox-onKeyPress

The onkeypress event occurs when the user presses a key (on the keyboard).

### TransactionCodeTextBox-onKeyUp

The onkeypress event occurs when the user presses a up-arrow key (on the keyboard).

### TransactionCodeTextBox-onFocus

The onfocus event occurs when an element gets focus.The onfocus event is most often used with input and select.
