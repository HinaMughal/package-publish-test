# `IBAN text box`
 
IBAN is a Text Box Component which is an internationally agreed account number identifying bank accounts across national borders to facilitate processing of transactions 

## Validation 
- Field length is 30
- Field is alphaNumeric

## Events Description
Note:  `None of events will occur if component is disabled`
### ibanTextBox-onChange

The onChange event occurs when the user change in text box

### ibanTextBox-onBlur

The onBlur event occurs before focus lost.

### ibanTextBox-onKeyPress

The onkeypress event occurs when the user presses a key (on the keyboard).

### ibanTextBox-onKeyUp

The onkeypress event occurs when the user presses a up-arrow key (on the keyboard).

### ibanTextBox-onFocus

The onfocus event occurs when an element gets focus.The onfocus event is most often used with input and select.
