# `Paying bank text box`

Paying Bank is a text box component which refers to the bank which is obliged to pay the beneficiary amount instructed by one to another party.

## Validation 
- Field is mandatory
- Field is disabled
- Maximum length is 54

## Events Description

### PayingBankTextBox-onChange

The onChange event occurs when the user change in text box

### PayingBankTextBox-onBlur

The onBlur event occurs before focus lost.

### PayingBankTextBox-onKeyPress

The onkeypress event occurs when the user presses a key (on the keyboard).

### PayingBankTextBox-onKeyUp

The onkeypress event occurs when the user presses a up-arrow key (on the keyboard).

### PayingBankTextBox-onFocus

The onfocus event occurs when an element gets focus.The onfocus event is most often used with input and select.
