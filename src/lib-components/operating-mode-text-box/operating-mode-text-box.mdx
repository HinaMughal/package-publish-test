# `Operating mode text box`

Operating Mode is text field component which details the account mode of operations for signing authorities.

## Validation 
- This field is mandatory
- Field is alphabetOnly
- Field is Disabled 
- Maximum Length is 2
- Field is left aligned

## Events Description

### OperatingModeTextBox-onChange

The onkeypress event occurs when the user presses a key (on the keyboard).
