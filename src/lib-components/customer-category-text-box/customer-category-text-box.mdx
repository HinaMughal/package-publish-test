# `Customer category text box`

Customer Category is a text box component referring to the type of customer at the time of account opening.

## Validation 
- Field is mandatory
- Field is left aligned
- Field is alphaSpecial
- Maximum length is 30

### CustomerCategoryTextBox-onChange

The onChange event occurs when the user change in text box

### CustomerCategoryTextBox-onKeyPress

The onkeypress event occurs when the user presses a key (on the keyboard).

The onkeypress event occurs when the user presses a up-arrow key (on the keyboard).

The onfocus event occurs when an element gets focus.The onfocus event is most often used with input and select.
